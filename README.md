# ⚔️ Gerador de Ficha de RPG - Losvenia

Este projeto foi desenvolvido como um desafio prático durante o **Curso de Python (Mundo 1)** do Professor Gustavo Guanabara (Curso em Vídeo).

## 📝 Sobre o Projeto
O programa simula a criação de um herói em um mundo fantástico chamado **Losvenia**. Ele coleta informações do jogador e gera atributos aleatórios, calculando valores essenciais para uma partida de RPG.

## 🚀 Tecnologias Utilizadas
- **Python 3**: Linguagem principal.
- **Módulo `random`**: Utilizado para simular a rolagem de dados (D20) para os atributos.

## 🛠️ Funcionalidades
- Coleta de Nome, Raça e Classe.
- Geração aleatória de 6 atributos principais (Força, Destreza, etc.).
- Cálculo automático de **Pontos de Vida (PV)** e **Carga Máxima**.
- Exibição de uma ficha formatada e organizada.
